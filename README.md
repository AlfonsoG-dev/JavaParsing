# Installation
1. Copy or clone the repository
```shell
git clone https://github.com/AlfonsoG-dev/JavaParsing
```
2. Build the project
- This project is build using my custom [build tool](https://github.com/AlfonsoG-dev/javaBuild).
```shell
javaBuild.exe --build
```
- Or use the *powersehll* script.
```shell
build.ps1
```
>- in Linux.
```shell
./build.sh
```
